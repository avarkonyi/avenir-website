- Root redirect: `/` -> `/hu`
- Nagybetus locale redirect: `/HU` -> `/hu` stb.
- Google es Bing indexelesi irany elinditva
- Bing site verification file
- Sitemap es robots alapok

- Hirek admin es publikus megjelenites
- Hirek tobbnyelvu publikalasa
- Hirkep feltoltes es publikus megjelenites
- Admin modulok alapjai:
  - News
  - Messages
  - Services
  - Positions
  - Partners
  - Certifications
  - Settings

### Meg Nyitott

## 2. Strategiai Irany: Avenir Operating System

Nem egyszeru weboldalt fejlesztunk tovabb, hanem egy belso mukodesi rendszert.

**Avenir Operating System =**

- publikus weboldal
- tartalomkezelo admin
- lead-kezelo mini-CRM
- OneDrive / SharePoint dokumentumtar
- ajanlat- es szerzodes-elokeszito folyamat
- SEO/GEO/AI keresore optimalizalt tartalomrendszer
- kesobb ugyfelportal es helyszini riporting

### Fo Elv

Eloszor a sajat folyamatot kell jol lemodellezni. Kulso CRM-et vagy Microsoft Graph integraciot csak akkor erdemes rakotni, ha mar pontosan latszik, hogyan dolgozik az Avenir a leadtol a szerzodesig.

## 5. Mini-CRM Strategia

### Mi legyen a CRM elso verzioja?

Nem kulso rendszer, hanem az Avenir admin Messages moduljanak bovese.

### Mi legyen egy lead?

Minden kapcsolatfelveteli uzenet potencialis lead.

Egy lead adatai:

- nev
- ceg
- email
- telefon
- szolgaltatasi erdeklodes
- uzenet
- nyelv
- erkezes datuma
- statusz
- felelos
- kovetkezo teendo
- becsult ertek
- ajanlat dokumentum link
- szerzodes dokumentum link
- belso megjegyzes

### Statusz pipeline

```text
Uj -> Felveve -> Minositett -> Ajanlat alatt -> Nyert
                                      \
                                       -> Elveszett
```

### Mi legyen kesobb?

- Outlook naptar emlekezteto
- Teams ertesites
- automatikus koveto email
- ajanlat PDF generalas
- CRM export
- Dynamics / HubSpot / Pipedrive integracio

## 6. OneDrive / SharePoint Strategia

### Elso lepes

Nem API integracio. Eloszor egységes dokumentumrendszer.

### Masodik lepes

Az adminban dokumentum URL mezok:

- proposalUrl
- contractUrl
- reportFolderUrl
- clientFolderUrl

### Harmadik lepes

Microsoft Graph integracio:

- ugyfelmappa automatikus letrehozas
- ajanlat PDF mentese
- szerzodes feltoltes
- havi riport mappa
- jogosultsagkezeles

### Nagyobb jovokep

Az Avenir adminbol indul a folyamat, de a dokumentumok a Microsoft 365 tenantban elnek. Igy az operativ mukodes nem ragad bele a webalkalmazasba, de a webalkalmazas tudja iranyitani es linkelni a dokumentumokat.

Keszobbi nagy modul:

- ugyfel belep
- latja a helyszineit

- uj igenyt kuld

Adminban belso tudastar:

- bemutatkozo szovegek
- tender valasz panelek
